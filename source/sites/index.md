## 资源导航

这里整理的是我自己长期会回访、也愿意推荐给别人的站点与资源。先保证少而精，后续再慢慢扩充。

## 建站与主题

### Hexo

- 官网：https://hexo.io/
- 适合想快速搭建静态博客的人，生态成熟，写作体验稳定。

### Butterfly

- 文档：https://butterfly.js.org/
- 我当前博客使用的主题，适合拿来做中度到重度定制。

### Cloudflare

- 官网：https://www.cloudflare.com/
- 常用来做 DNS、CDN、基础安全和缓存优化。

## 运维与部署

### Docker

- 官网：https://www.docker.com/
- 服务打包和环境统一的第一选择，部署时很省心。

### Nginx

- 官网：https://nginx.org/
- 静态资源分发、反向代理和 HTTPS 配置都很常用。

### DigitalOcean

- 官网：https://www.digitalocean.com/
- 适合作为轻量博客或小服务的 VPS 选择之一。

## 学习与文档

### MDN Web Docs

- 地址：https://developer.mozilla.org/
- 前端和 Web 基础文档非常扎实，查语法和 API 都很顺手。

### Git 官方文档

- 地址：https://git-scm.com/doc
- Git 命令和工作流细节，还是官方文档最靠谱。

### LinuxCommand

- 地址：https://linuxcommand.org/
- 适合补 Linux 命令行基础。
