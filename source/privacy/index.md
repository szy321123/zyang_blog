# 隐私政策 (Privacy Policy)

欢迎访问 **「汤圆是只乳白」**。本站尊重并重视访问者隐私，本页面用于说明本站可能收集的信息类型、使用方式以及相关保护措施。

## 1. 我们收集的信息

**1.1 自动收集的访问信息**
为了提供更稳定的访问体验，本站及所使用的第三方服务可能会在访问过程中记录部分匿名访问数据，包括但不限于：
- 您的 IP 地址、浏览器类型及版本；
- 操作系统、设备类型信息；
- 访问本站的具体时间、页面浏览量、停留时间及来源链接。

以上信息主要用于访问统计、性能分析与安全防护，不会单独用于识别您的真实身份。

**1.2 您主动提供的信息**
当您在本站评论、留言或通过邮件联系我时，您可能会主动提供以下信息：
- 昵称 / 姓名
- 电子邮箱地址
- 个人网站链接
- 评论内容

这些信息仅用于互动交流、问题回复与评论展示，不会对外出售、出租或用于未经授权的商业推广。

## 2. Cookies 的使用

本站可能使用 Cookies 或类似技术以提升访问体验，例如记录主题偏好、缓存部分页面状态或保存评论表单信息。部分第三方服务也可能独立使用 Cookies。您可以通过浏览器设置清除或禁用 Cookies，这通常不会影响基础阅读功能。

## 3. 第三方链接与服务

本站文章中可能包含外部链接，或接入用于统计、加速、评论等功能的第三方服务。当您离开本站或与第三方服务交互时，其隐私处理规则将不再受本协议约束，建议您在提供个人信息前自行查阅对应平台的隐私政策。

## 4. 数据安全

本站会采取合理且必要的安全措施（如 HTTPS 加密传输）保护数据，尽力降低信息泄露、损坏或丢失的风险。但互联网环境并不存在绝对安全，本站无法对所有传输过程作出 100% 安全保证。

## 5. 您可以如何管理个人信息

您可以根据需要采取以下方式管理相关信息：
1. 在浏览器中清除或禁用 Cookies；
2. 在不留言、不评论、不发邮件的情况下匿名浏览公开内容；
3. 通过页面底部提供的联系方式就隐私相关问题与我联系。

## 6. 协议更新

本隐私协议可能会根据站点功能调整而更新，更新版本将直接发布在本页面。您继续访问本站，即视为接受更新后的协议内容。

## 联系方式

如果您对本站的隐私处理方式有疑问或建议，可通过页面底部提供的邮箱或 GitHub 与我联系。
